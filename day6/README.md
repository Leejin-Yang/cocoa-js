# To Do List Web

입력 받은 task를 추가하는 간단한 투두 리스트를 만든다.

추가, 완료, 삭제의 기능을 가진다.

<br>

## 학습 체크리스트

- JS와 DOM과의 상호작용 경험.

- DOM 노드 탐색하고 추가하는 API를 안다.

- Event를 등록하고 Event listenter를 등록할 수 있다.
